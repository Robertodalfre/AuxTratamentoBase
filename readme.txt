# JS no pelo

Este repositório contém dois projetos em JavaScript:

1. **Remover Bloco de Texto**
   - Descrição: Remove um bloco de texto específico de um arquivo.
   - Arquivo: `removerBloco.js`

2. **Gerador de Script SQL**
   - Descrição: Gera um script SQL com base nos arquivos dentro de uma determinada pasta.
   - Arquivo: `geraScriptSql.js`

## Como Executar os Projetos

### Pré-requisitos
Certifique-se de ter o Node.js instalado em sua máquina.

### Instalação
```bash
npm install
